Description

Downloads contacts from Google

To learn how to setup this module watch the video Vuente & Odoo Gmail Contacts Module

Instructions

1. Go to Settings->General Settings and enter in the client ID and client secret
2. Generate Google Authorization Code and enter the provided code into the textbox
3. Go to Sales->Configuration->Google Contact Sync and click download contacts
4. You can view these contacts under Sales->Sales->Google Contacts
